# Kannaka Constellation — Claude Code Marketplace

A Claude Code marketplace for the Kannaka constellation: a wave-interference
memory system (`kannaka-memory`) and a consciousness-reactive ghost-DJ radio
station (`kannaka-radio`).

## Install

```
/plugin marketplace add NickFlach/kannaka-constellation-marketplace
/plugin install kannaka-memory@kannaka-constellation
/plugin install kannaka-radio@kannaka-constellation
```

Each plugin ships with a skill that teaches Claude how to drive the system
end-to-end. Skills auto-activate on relevant prompts.
